# Vibe Tasks

Учебный проект для освоения GitHub с нуля и работы с AI (Codex).
Цель: научиться создавать репозитории, вести задачи через Issues и делать изменения через ветки + Pull Request.

## Что будет в проекте
- [ ] Добавление задачи
- [ ] Просмотр списка задач
- [ ] Отметка "выполнено"
- [ ] Базовые тесты
- [ ] CI (GitHub Actions)

## Структура (появится по мере обучения)
- `src/` — код приложения
- `tests/` — тесты
- `docs/` — документация
- `.github/` — шаблоны Issue/PR и Actions

## Как работать (процесс)
1) Создаю Issue (задачу)
2) Создаю ветку под задачу
3) Делаю изменения
4) Открываю Pull Request
5) Проверяю и сливаю изменения (merge)

## Статус
Проект в разработке. Первый этап: настройка GitHub и структуры репозитория.
