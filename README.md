# Vibe Tasks

Учебный проект для освоения GitHub с нуля и работы с AI (Codex).
Цель: научиться создавать репозитории, вести задачи через Issues и делать изменения через ветки + Pull Request.

## Что будет в проекте
- [ ] Добавление задачи
- [ ] Просмотр списка задач
- [ ] Отметка "выполнено"
- [ ] Базовые тесты
- [ ] CI (GitHub Actions)

## План обучения (7 шагов)

- [ ] 1) Настроить GitHub: профиль, безопасность (2FA)
- [ ] 2) Оформить репозиторий: README, LICENSE, .gitignore
- [ ] 3) Научиться задачам: создать Issues и (по желанию) Project
- [ ] 4) Научиться веткам: делать правки в отдельной ветке
- [ ] 5) Научиться Pull Request: описание, комментарии, merge (squash)
- [ ] 6) Подготовить Windows для реальной работы: установить Git for Windows и настроить доступ (HTTPS/SSH)
- [ ] 7) Подключить базовую автоматизацию: GitHub Actions (CI) + релиз (tag/release)


## Структура (появится по мере обучения)
- `src/` — код приложения
- `tests/` — тесты
- `docs/` — документация
- `.github/` — шаблоны Issue/PR и Actions

## Как работать (процесс)
1) Создаю Issue (задачу)
2) Создаю ветку под задачу
3) Делаю изменения
4) Открываю Pull Request
5) Проверяю и сливаю изменения (merge)
